Mastering Shopify 2.0: Building Custom Sections with JSON Templates

Shadrack Wangai Maina

Shopify Expert & Full Stack Developer

December 15, 2023 12 min read

Introduction

Shopify 2.0 introduced a revolutionary approach to theme development with JSON templates and enhanced section capabilities. This guide will walk you through creating powerful, reusable custom sections that merchants can easily customize without touching code.

Understanding Shopify 2.0 Architecture

The core of Shopify 2.0 lies in its flexible section-based architecture. Unlike the traditional theme structure, Shopify 2.0 allows sections to be added anywhere on any page using JSON templates. This provides unprecedented flexibility for merchants.

Key Features of Shopify 2.0:

  • Section Groups: Organize sections into logical groups for better content management
  • App Blocks: Third-party apps can now inject content into any section
  • Metafields: Enhanced support for dynamic content from metafields
  • JSON Templates: Complete page structure defined in JSON format

Building Your First Custom Section

Let's create a flexible hero section with multiple layout options, dynamic content, and full customization capabilities. This section will demonstrate advanced schema features and Liquid templating.

sections/custom-hero.liquid 
{% schema %}
{
  "name": "Custom Hero Section",
  "settings": [
    {
      "type": "select",
      "id": "layout",
      "label": "Layout Style",
      "options": [
        { "value": "full", "label": "Full Width" },
        { "value": "split", "label": "Split Layout" },
        { "value": "overlay", "label": "Image Overlay" }
      ],
      "default": "full"
    },
    {
      "type": "image_picker",
      "id": "hero_image",
      "label": "Hero Image"
    },
    {
      "type": "text",
      "id": "heading",
      "label": "Heading",
      "default": "Welcome to Our Store"
    }
  ]
}
{% endschema %}

Advanced Schema Configuration

The schema is where the magic happens. It defines all customizable settings that merchants can control through the theme editor. Understanding schema types and their capabilities is crucial for creating flexible sections.

Best Practices for Schema Design:

  1. Group related settings using headers and info blocks
  2. Provide sensible defaults for all settings
  3. Use clear, descriptive labels and help text
  4. Leverage conditional settings to reduce complexity
  5. Test thoroughly with different content lengths

Performance Optimization Tips

Custom sections should be optimized for performance. Here are key strategies to ensure your sections load quickly and don't impact Core Web Vitals:

  • Lazy load images using native loading="lazy" attribute
  • Minimize inline JavaScript; use theme.js for shared functionality
  • Implement CSS containment for better rendering performance
  • Use Shopify's image filters for responsive images
  • Avoid unnecessary Liquid loops and complex logic

Conclusion

Mastering Shopify 2.0's section architecture opens up endless possibilities for creating flexible, merchant-friendly themes. By following these best practices and understanding the power of JSON templates, you can build themes that are both powerful and easy to customize.
